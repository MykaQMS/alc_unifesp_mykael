📘 Álgebra Linear Computacional — UNIFESP SJC

Este repositório foi criado com o objetivo de armazenar e organizar as atividades desenvolvidas em Python ao longo da disciplina Álgebra Linear Computacional, oferecida pela Universidade Federal de São Paulo (UNIFESP) — campus São José dos Campos.

🎓 Sobre a disciplina

A disciplina aborda conceitos fundamentais de álgebra linear com foco computacional, incluindo:

Operações com matrizes e vetores
Sistemas lineares
Posto e independência linear
Autovalores e autovetores
Aplicações numéricas com Python

As atividades práticas são implementadas utilizando bibliotecas como:

NumPy
SciPy
Matplotlib (quando aplicável)
